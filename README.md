Real Estate Conversion Dashboard

An interactive analytics dashboard built using Python and Streamlit to monitor real estate conversion performance, property trends, revenue insights, and market behavior.

📌 Project Overview

This project transforms raw real estate transaction data into actionable business insights through an interactive dashboard.

The dashboard focuses on the complete conversion funnel:

Listing → Visit → Booking → Move-In

It helps analyze:

Conversion performance
Property demand
Revenue trends
City-wise market insights
Closing efficiency
