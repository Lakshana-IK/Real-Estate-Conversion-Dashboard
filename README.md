# 🏠 Real Estate Conversion War Room Dashboard

An interactive analytics dashboard built using Python and Streamlit to monitor real estate conversion performance, revenue trends, property demand, and market behavior.

---

# 📌 Project Overview

This project transforms raw real estate transaction data into actionable business insights through an interactive dashboard.

The dashboard focuses on the complete customer conversion funnel:

Listing → Visit → Booking → Move-In

The application helps analyze:
- Conversion performance
- Revenue trends
- Property demand
- City-wise market insights
- Closing efficiency

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/real-estate-conversion-dashboard.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd real-estate-conversion-dashboard
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Dashboard

```bash
python -m streamlit run app.py
```
